Claude Code Usage Tracker

VSCode拡張機能でClaude Codeの使用量とコストをリアルタイム追跡

📦 GitHub 🛒 VS Code Marketplace ❤️ スポンサーになる

主な機能

📊

リアルタイム追跡

Claude Codeの会話をリアルタイムで監視し、トークン使用量とコストを自動計算

💰

詳細なコスト分析

Input/Output/Cache別のトークン数と、各Claudeモデルの正確なコスト計算

📈

包括的な分析

日次・週次・月次レポートと利用傾向の分析、$300プランの価値評価

🔍

セッション分析

プロジェクト別の使用量、個別会話の詳細追跡、効率性の可視化

⚡

高性能

効率的なファイル監視、低リソース使用、大量JSONLファイルの高速解析

🛠️

環境対応

Local/WSL/Container環境の自動検出、カスタム設定対応

スクリーンショット

メインダッシュボード

今日の利用状況と全体統計を一目で確認

利用状況分析

月間平均との比較と$300プランの価値評価

詳細レポート

日次・週次・月次の詳細な使用量レポート

インストール方法

1

VS Code Marketplaceからインストール

VS Code内で「Claude Code Usage Tracker」を検索してインストール

ext install your-publisher.ccusage-ext
2

自動検出

拡張機能がClaude Codeのプロジェクトディレクトリを自動検出

3

ダッシュボードを開く

コマンドパレット（Ctrl+Shift+P）から「Claude Usage: Show Usage Dashboard」を実行

使用方法

📊 メインダッシュボード

今日の利用状況、全体統計、利用状況分析を表示

Ctrl+Shift+P → "Claude Usage: Show Usage Dashboard"

📅 日次レポート

指定日の詳細な使用量とセッション情報

Ctrl+Shift+P → "Claude Usage: Daily Report"

📆 月次レポート

月別の集計データと日別内訳

Ctrl+Shift+P → "Claude Usage: Monthly Report"

🔴 ライブセッション

現在アクティブな会話をリアルタイム監視

Ctrl+Shift+P → "Claude Usage: Live Session Monitor"
